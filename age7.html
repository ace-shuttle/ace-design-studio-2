Scratch

Scratch is developed by MIT experts to introduce children to coding concepts through visual coding. Scratch has a colourful and drag-and-drop interface which makes coding easy for young learners. We have designed a course to be fun and engaging for beginners to get started with coding. Join us to learn how to create your own flappy bird inspired game using scratch!
